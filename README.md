
# Simple To Do List

A simple project made to recreate a to do list using react and redux.


## Installation

Clone the project

```bash
  git clone https://github.com/alektobias/to-do-list
```

Install dependencies:

```bash
  yarn install -f
  npm install -f
```

And Finally run it as development mode:

```bash
  yarn dev
  npm run dev
```
